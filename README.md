# NSE Stock API

Get ticker price of any stock from NSE.

<br>

### Install

Vercel Deploy

#### Manual

1. ```npm i -g nse-api```
2. ```nseapi```

<br>

### Methods

#### Request stock price

```
https://localhost:3000/<stock-symbol>
Example: https://example.vercel.app/IRCTC
```
